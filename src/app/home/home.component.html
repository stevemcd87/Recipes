Technologies

  • Angular CLI version 8.3.8.
  • Angular's InMemoryDataService to mimic communication with a remote data server.
  • Auth0 for user authentication.

How to Use

  • Logging in will allow the user to add a recipe to the list of recipes.
  • By clicking ''Posts'', on the top Nav bar, you will be navigated to a list of posts/recipes previously created.
  • You can then click on an individual recipe to see how the recipe will look on a mobile phone.
  • The user can also have the option to edit or delete the post if they're the author of the post. Mobile phone will be displayed while editing post to see how updated post will look in mobile view.
